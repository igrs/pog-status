# POG Status

POG登録馬の状態や氏名馬の獲得総賞金を表示します。(from [netkeiba](http://www.netkeiba.com))

## 実行方法

```$ pog_status.exe sample.txt```  
(pathを通してもよいです。ファイル名は任意です。)

## ファイル

netkeibaの馬のIDを羅列します。  
ファイル名は任意です。
```
2013106025
2013106101
・
・
```

## 出力情報

* 個別

    - 馬名
    - 中央獲得賞金
    - 地方獲得賞金
    - 戦績

* 合計
　　  
    - 中央獲得賞金合計
    - 地方獲得賞金合計
    - 総獲得賞金
    - 勝ち上がり率（新馬未勝利脱出頭数/全頭数）
    - デビュー率（デビュー頭数/全頭数）
    - 獲得賞金の最も多い馬名
